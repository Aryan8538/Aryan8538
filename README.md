A full-stack MERN application that combines real-time market data with an AI financial advisor, enabling users to learn investing through simulated, risk-free paper trading.

| Category | Details |
|---|---|
| **Stack** | MongoDB, Express, React, Node.js, Groq LLM API |
| **Scale** | Multi-user platform with persistent portfolios and live market watchlists |
| **Performance** | Optimized concurrent API calls to prevent single-symbol failures from blocking the full watchlist |
| **Security** | Session-based authentication with isolated per-user data stores |
| **Impact** | Enables risk-free financial literacy through simulated investing backed by real market data |
| **Repository** | [github.com/aryan-devbuilds/finsight-ai](https://github.com/aryan-devbuilds/finsight-ai) |

FinSight AI is my flagship project — engineered to demonstrate real production debugging depth rather than surface-level feature building. Development involved diagnosing and resolving critical authentication regressions, recovering from upstream LLM model deprecations, and hardening error handling across market data and advisory modules.
